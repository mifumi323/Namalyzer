●概要
溜め込んだアクセスログを後でゆっくり見るための、ちょっとはマシな
アクセス解析です。
動作速度にさほどこだわらず、特に加工されていない生ログを読み込んで
解析するためこの名前がついています。
combined形式のアクセスログを解析することができます。
また、動作確認をするつもりはありませんが、一応common形式にも
対応しているはずです。

のーんびりゆったり。あわてなーい


●動作環境
.NET Framework 4.7.2が動く環境
以下の環境で動作確認済みです。
